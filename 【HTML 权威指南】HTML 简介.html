元素

开始标签    内容    结束标签
    |       |        |
<code>   示例内容   </code>

注：元素标签不区分大小写，但习惯小写；

  • 常用元素
元素 说明 章节
DOCTYPE HTML 文档开始 7
html HTML 文档 html 部分 7
head HTML 文档头 7
title HTML 文档标题 7
body HTML 文档内容 7
style CSS 样式 7
a 超链接 8
code 计算机代码片段 8
input 用户输入 8
hr 主题 9
p 段落 9
table 表格组织 11
th 表头单元格 11
td 表格单元格 11
tr 表格行 11
button 按钮 12
label 元素说明标签 12
textarea 输入文本框 14
  • 空元素
<code></code>
  • 自闭合标签
<code />
效果同上；
  • 虚元素
<hr />
<br />
建议使用自闭合标签标示，虽然只使用开始标签亦可；

元素属性

<a href="https://www.baidu.com">百度一下</a>

href：定义的元素属性，表示超链接目标 URL

注：属性用在开始标签内部，通常由属性名与属性值组成；

  • 多个属性

使用空格分离即可；

<a href="https://www.baidu.com" id="firstLink">百度一下</a>
  • 布尔属性

可省略属性值；

<button>可用的按钮</button>
<button disabled>不可用的按钮</button>

以下亦可
<button disabled=true>不可用的按钮</button>
<button disabled="">不可用的按钮</button>
  • 自定义属性

使用 data- 开头命名；

创建 HTML 文档

  • 外层结构
<!DOCTYPE html>
<html>

    <!-- 各种元素 -->

</html>
  • 元数据

描述 HTML 文档的一些信息，包含在 <head> 元素内部；

<!DOCTYPE html>
<html>
    <head>
    
        <!-- 描述 -->
        
    </head>
</html>
  • 内容

文档主体部分，<body> 元素内部；

<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>

        <!-- 内容 -->

    </body>
</html>
  • 元素关系

父元素：包含关系
子元素：被包关系仅下一层
后代元素：被包含关系所有
兄弟元素：同级

  • 元素类型

元数据元素：构建 HTML 基本结构，以及补充信息
流元素：确定元素合法父子元素的范围
短语元素：特殊的流元素

全局属性

配置所有元素的公共属性；

  • accesskey

设置选择元素快捷键；

<form>
    <input accesskey="k" />
</form

windows 上是 Alt+"k"；mac 没试出来；

  • class

设置元素归类，应用 CSS 样式；

<style type="text/css">
class1{
    color:red;
}
</style>

<p class="class1">公开课</p>
  • contenteditable

可修改页面内容；

<p contenteditable>公开课</p>
  • contextmenu

快捷菜单；

  • dir

文字方向；

<p dir="rtl">公开课</p>
<p dir="ltl">公开课</p>
  • draggable

元素是否可拖放；

37 章
  • dropzone

拖放相关；

37 章
  • hidden

隐藏；

<p hidden>公开课</p>
<p >公开课</p>
  • id

唯一标识；

<style type="text/css">
#lesson{
    color:red;
}
</style>

<p id="lesson">公开课</p>

还可以作为锚点跳转；

  • lang

说明元素使用的语言；

<html lang="en">
<p lang="zh-CN">公开课</p>
  • spellcheck

拼写检查；

<textarea spellcheck>lessn</textarea>
  • style

元素 CSS 样式；

<p style="color:red">公开课</p>
  • tabindex

Tab 元素切换；

<form>
    <label>Name: <input type="text" tabindex="1"/></label>
    <label>Age: <input type="text" tabindex="2"/></label>
    <label>:Phone <input type="text" tabindex="-1"/></label>
</form>
  • title

元素额外信息；一般用于工具提示等；

<head>
    <title>认识html标签</title>
</head>

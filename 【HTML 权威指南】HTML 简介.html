一、元素

1. 结构示例：

开始标签    内容    结束标签
    |       |        |
<code>   示例内容   </code>
  • 举例：
<p>Hello World</p>

2. 本章用到的元素

  • 略；

3. 空元素

  • 只有元素标签，没有内容；
<code></code>

4. 自闭合标签

  • 开始、结束标签二合一；
正常：<a href="tets.html">链接</a>
自闭合：<a href="tets.html"/>链接

还是统一使用正常的好，看着舒服。

5. 虚元素

  • 只有开始标签；
<hr>
<br>
<img src="test.png">

二、元素属性

  • 属性用在开始标签内部，通常由属性名与属性值组成；

1. 结构示例：

    属性名称   属性值
      |        |    
<code key = 'value'>   示例内容   </code>
  • 举例超链接：
<a href="https://www.baidu.com">百度一下</a>

2. 多个属性

  • 使用空格分离即可；
<a href="https://www.baidu.com" id="firstLink">百度一下</a>

3. 布尔属性

  • 可省略属性值；
<button>可用的按钮</button>
<button disabled>不可用的按钮</button>

以下亦可
<button disabled=true>不可用的按钮</button>
<button disabled="">不可用的按钮</button>

4. 自定义属性

  • 使用 data- 开头命名；
<a href="https://www.baidu.com" id="firstLink" data-testName="百度">百度一下</a>

三、创建 HTML 文档

  • 创建文件，如 test.html；
  • 如果是 webStorm 那么更方便；

1. 外层结构

  • 标识文档本身是 HTML 文档；
<!DOCTYPE html>
<html>

    <!-- 各种元素 -->

</html>

2. 元数据与内容

  • 元数据：描述 HTML 文档信息，<head> 元素内部；
  • 内容：展示用户界面等，<body> 元素内部；
<!DOCTYPE html>
<html>
    <head>
    
        <!-- 描述 -->
        
    </head>
    <body>

        <!-- 内容 -->

    </body>
</html>

3. 元素关系

  • 父元素：包含关系
  • 子元素：被包关系仅下一层
  • 后代元素：被包含关系所有
  • 兄弟元素：同级

4. 元素类型

  • 元数据元素：构建 HTML 基本结构，以及补充信息
  • 流元素：确定元素合法父子元素的范围
  • 短语元素：特殊的流元素

四、HTML 实体

  • 特殊字符的的显示，使用实体编号；
字符 实体名称 实体编号
< &lt &#60
> &gt &#62
& &amp &#30
€ &euro &#8364
£ &pound &#163
§ &sect &#167
© &copy &#169
® &reg &#174
™ &trade &#8482
  • 举例：
说了是村长 &copy; 2017
说了是村长 &#169; 2017

五、全局属性预览

  • 配置所有元素的公共属性；

1. accesskey

  • 设置选择元素快捷键；
<form>
    <input accesskey="k" />
</form

windows 上是 Alt+"k"；mac control+option+"K"；

2. class

  • 设置元素归类，应用 CSS 样式；
<style type="text/css">
class1 {
    color:red;
}
</style>

<p class="class1">公开课</p>

3. contenteditable

  • 可修改页面内容；
<p contenteditable>公开课</p>

4. contextmenu

  • 快捷菜单；当时暂不支持；

5. dir

  • 文字方向；
<p dir="rtl">公开课</p>
<p dir="ltl">公开课</p>

6. draggable 和 dropzone

  • 元素拖放相关，第五部分【高级功能】详细讲解；

7. hidden

隐藏；

<p hidden>公开课</p>
<p >公开课</p>

8. id

  • 唯一标识；
<p id="lesson">公开课</p>

9. lang

  • 说明元素使用的语言；
<html lang="en">
<p lang="zh-CN">公开课</p>

10. spellcheck

  • 拼写检查；
<textarea spellcheck>lessn</textarea>

11. style

  • 元素 CSS 样式；
<p style="color:red">公开课</p>

12. tabindex

  • Tab 按键元素切换；
<form>
    <label>Name: <input type="text" tabindex="1"/></label>
    <label>Age: <input type="text" tabindex="2"/></label>
    <label>:Phone <input type="text" tabindex="-1"/></label>
</form>

13. title

  • 元素额外信息；一般用于工具提示等；
<head>
    <title>认识html标签</title>
</head>

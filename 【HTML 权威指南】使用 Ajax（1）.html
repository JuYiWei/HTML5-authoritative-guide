5.1 使用 Ajax（1）

  • 一、Ajax 起步
      • 1. 处理响应
  • 二、使用 Ajax 事件
  • 三、错误处理
  • 四、获取和设置标头
      • 1. 覆盖请求的 HTTP 方法，设置 head 需要在 open 之后；
      • 2. 禁用内容缓存
      • 3. 读取响应标头
  • 五、生成跨源 Ajax 请求
  • 六、中止请求

一、Ajax 起步

 function getInfo() {
            var httpRequest = new XMLHttpRequest();
            httpRequest.onreadystatechange = handleResponse;
            httpRequest.open("GET","url");
            httpRequest.send();
        }

        function handleResponse(e) {
            if (e.target.readyState == XMLHttpRequest.DONE && e.target.status == 200) {
                document.getElementById("content").innerHTML("request OK");
            } else {
                document.getElementById("content").innerHTML("request error");
            }
        }

1. 处理响应

readyStatus 值 描述
UNSENT 已创建
OPENED 已 open
HEADERS_RECEIVED 已收到标头
LOADING 收到响应
DONE 响应完成或失败

还需要根据 status 判断 HTTP 请求的状态码，确定请求结果；

二、使用 Ajax 事件

事件名称 描述
abort 请求中止
error 请求失败
load 请求成功
loadend 请求完成（不管成功和失败）
loadstart 请求开始
progress 触发提示进度
readystatechange 请求生命周期不同阶段
timeout 超时
  • 以上事件，除 readystatechange 之外，都为 ProgressEvent；
ProgressEvent 额外属性 描述
lengthComputable bool，是否可以计算数据流长度
loaded 已载入数据量
total 总数据量

三、错误处理

  • 第一类错误：某些因素阻止请求发送到服务器；

  • 第二类错误：响应内容不为预期；

    1. 处理设置错误；
            try {
                httpRequest.open("GET","http://apis.baidu.com/heweather/pro/attractions");
                httpRequest.send();
            } catch (err) {

            }
  • 2. 处理请求错误；
            httpRequest.onerror = function () {
                // do
            };
  • 3. 处理应用程序错误；
            if (e.target.status == 200) {
                    // ok
            } else {
                // error
            }

四、获取和设置标头

方法 描述
setRequestHeader(
,)
 指定值设置标头
getResponseHeader(
)
 获取指定标头
getAllResponseHeaders() 以单个字符串形式获取所有标头

1. 覆盖请求的 HTTP 方法，设置 head 需要在 open 之后；

            try {
                httpRequest.open("GET","http://apis.baidu.com/heweather/pro/attractions");
                httpRequest.setRequestHeader("apikey","87ac51ed29c3c53d37bb1e0a77996731");
                httpRequest.send();
            } catch (err) {

            }

2. 禁用内容缓存

                httpRequest.setRequestHeader("Cache-Control","no-cache");

3. 读取响应标头

            if (e.target.readyState == HEADERS_RECEIVED) {
                var contentType = httpRequest.getResponseHeader("Content-Type");
                console.log(contentType);
            }

五、生成跨源 Ajax 请求

  • 请求接口添加标头：'Access-Control-Allow-Origin:*' 允许所有源；

六、中止请求

httpRequest.abort();

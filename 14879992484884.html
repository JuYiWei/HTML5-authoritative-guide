第七章 创建 HTML 文档

基本文档结构

<!DOCTYPE html>
<html>
    <head>
        // head 部分
    </head>
    <body>
        // 内容部分
    </body> 
</html>

DOCTYPE

  • 表示处理的是 HTML 文档；
  • 标记文档内容 HTML 版本；
  • 放在文档第一行；

html

  • 文档根元素，包含内容和元素；

head

  • 包含文档元数据，提供文档相关信息；

body

  • 文档内容部分；

元数据说明文档

  • <head>标签中，设置；

设置文档标题

  • 通常显示为浏览器标签；
<head>
    <title>说了是村长</title>
</head>

设置相对 URL 基准

<head>
    <base href="http://testurl/">
</head>

元数据说明文档

  • 指定名/值元数据对（可以设置多个）
<head>
    <meta name="author" content="说了是村长">
</head>
  • 声明字符编码
<head>
    <meta charset="UTF-8">
</head>
  • 模拟 HTTP 标头字段
<head>
    <meta http-equiv="refresh" content="5">
</head>

http-equiv 有三个值可用：refresh、default-style、content-type；

CSS 样式

<style type="text/css" media="screen AND (min-width:500px">
   p {
       color: red;
   }
</style>
  • type：总是“text/css”；
  • scoped：指定范围，基本没用；
  • media：设备选择与适用条件
设备 说明
all 所有
aural 语音合成器
braille 盲文设备
handheld 手持设备
projection 投影机
print 打印
screen 显示器
tty 电传打字机
tv 电视机
条件 说明
width / height 浏览器宽高
device-width / device-height 设备宽高
resolution 像素密度
orientation 设备朝向：portrait、landscape
aspect-ratio / device-aspect-ratio 浏览器宽高比
color / monochrome 彩色、黑白设备像素占有的二进制数
color-index 设备显示颜色数目
scan 电视扫描模式：progressive、interlace
grid 网格显示内容：0、1

指定外部资源

<link rel="stylesheet" href="https://res.wx.qq.com/open/libs/weui/0.4.2/weui.css"/>
属性 说明
href 资源路径
hreflang 资源语言
media 设备，与上面相同
rel 资源关联类型
sizes 指定图标大小
type 关联资源 MIME 类型
rel 值 说明
alternate 文档替代版本
author 文档作者
help 文档的说明文档
icon 图标资源
license 许可证
pingback 回探服务器
prefetch 预先获取资源
stylesheet 载入外部样式表
  • 载入样式表
<link rel="stylesheet" href="test.css">
  • 定义网站标志
<link rel="icon" href="123.jpg" type="image/jpg">
  • 预先获取资源
<link rel="prefetch" href="/page2.html">

脚本元素

建议放在 head 部分，便于维护；

script

  • 文档内部脚本，type 默认 JavaScript；
<script type="javascript">
    document.write("Hello");
</script>
  • 连接外部脚本
<script src="test.js"></script>
  • 脚本延迟执行 defer
<script defer src="test.js"></script>
  • 脚本异步执行 async
<script async src="test.js"></script>

noscript

  • 用来对禁用的 JavaScript 或不支持 JavaScript 的浏览器提示内容；
<noscript>
    JavaScript is required！
</noscript>

用路径绘制

路径方法 说明
beginPath() 开始
closePath() 闭合
fill() 填充
isPointInPath(x,y) 判断
lineTo(x,y) 绘制子路径
moveTo(x,y) 移动到
rect(x,y,w,h) 矩形
stoke() 描边
  • 绘制直线
    var ctx = document.getElementById("myCanvas").getContext("2d");

    ctx.fillStyle = "yellow";
    ctx.strokeStyle = "red";

    ctx.beginPath();
    ctx.moveTo(100,100);
    ctx.lineTo(100,150);
    ctx.lineTo(200,150);
    ctx.closePath();
    ctx.fill();
    ctx.stroke();
  • 绘制矩形
    var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.fillStyle = "yellow";
    ctx.strokeStyle = "red";
    ctx.rect(100,100,100,100);
    ctx.fill();
    ctx.stroke();

绘制圆弧

方法 说明
arc(x,y,rad,startAngle,endAngle,direction) 扇形：位置，半径，起末角度，方向
arcTo(x1,y1,x2,y2,rad) 圆弧
  • arcTo，3 点画圆弧
    var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.strokeStyle = "red";

    ctx.beginPath();
    ctx.moveTo(100,10);
    ctx.arcTo(200,10,200,110,100);

    ctx.stroke();
  • arc
    var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.strokeStyle = "red";

    ctx.beginPath();
    ctx.arc(70,70,60,0,Math.PI * 3 / 4, true);
    
    ctx.stroke();

绘制贝塞尔曲线

方法 说明
bezierCurveTo(cx1,cy1,cx2,cy2,x,y) 三次
quadraticCurveTo(cx,cy,x,y) 二次
  • 三次 bezierCurveTo
   var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.strokeStyle = "red";

    ctx.beginPath();
    ctx.moveTo(100,100);
    ctx.bezierCurveTo(110,120,120,80,200,100);
    
    ctx.stroke();
  • 二次 quadraticCurveTo
    var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.strokeStyle = "red";

    ctx.beginPath();
    ctx.moveTo(100,100);
    ctx.quadraticCurveTo(110,120,200,100);

    ctx.stroke();

创建剪辑区域

    var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.strokeStyle = "red";

    ctx.beginPath();
    ctx.rect(0,0,100,100);
    ctx.fill();

    ctx.beginPath();
    ctx.rect(100,100,100,100);
    ctx.clip();

    ctx.rect(0,0,100,100);
    ctx.stroke();

绘制文本

方法
fillText(,x,y,width)
strokeText(,x,y,width)
属性 说明
font
textAlign
textBaseline 
    var ctx = document.getElementById("myCanvas").getContext("2d");

    ctx.font = "36px sans-serif";
    ctx.strokeText("Hello",100,100);

    ctx.fillText("Hello", 100,150);
    ctx.stroke();

使用特效和变换

  • 阴影
阴影属性 说明
shadowBlur 模糊
shadowColor 颜色
shadowOffsetX 偏移 x
shadowOffsetY 偏移 y 
    var ctx = document.getElementById("myCanvas").getContext("2d");
    ctx.font = "36px sans-serif";

    ctx.shadowBlur = 5;
    ctx.shadowColor = "red";
    ctx.shadowOffsetX = 2;
    ctx.shadowOffsetY = 10;

    ctx.fillText("Hello", 100,150);
    ctx.stroke();
  • 透明度
    ctx.globalAlpha = 0.6;
  • 合成，挺复杂的，未尝试
globalCompositeOperation 值 说明
copy 绘制来源（忽略透明度）
destination-atop
destination-in
destination-over
destination-out
lighter
source-atop
source-in
source-out
source-over
xor
  • 变换
属性 说明
scale(,) 缩放
rotate() 旋转
translate(,) 平移
transform(a,b,c,d,e,f) 合并现有变换矩阵
setTransform(a,b,c,d,e,f) 替换现有矩阵变化 
    var ctx = document.getElementById("myCanvas").getContext("2d");

    ctx.rotate(0.5);

    ctx.fillText("Hello", 100,150);
    ctx.stroke();

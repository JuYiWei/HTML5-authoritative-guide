其他表单元素

选项列表

  • select、option一般选择；
name 名称
disable 禁用
form 表单
size 选择框大小，行数
multiple 是否多选，control+/command+
autofocus 自动聚焦
required 必须 
<label>地址：
    <select name="address" size="5" multiple>
        <option value="hangzhou" label="hangzhou"></option>
        <option value="shanghai" label="shanghai"></option>
    </select>
</label>
  • optgroup可选项分组；
<label>地址：
    <select name="address" size="5" multiple>
        <optgroup label="浙江省">
            <option value="杭州" label="杭州"></option>
            <option value="嘉兴" label="嘉兴"></option>
        </optgroup>
        <optgroup label="上海">
            <option value="上海" label="上海"></option>
        </optgroup>
    </select>
</label>

多行输入

  • textarea属性：name、disable、form、readonly、maxlength、autofocus、required、placeholder、dirname、rows、wrap、cols；
<textarea name="message" cols="20" rows="5" wrap="hard" id="message">

计算结果

  • output

<form oninput="result.value=a.valueAsNumber * b.valueAsNumber">
    <input type="number" id="a" name="a"> * <input type="number" id="b" name="b"> =
    <output name="result"></output>
</form>

公私钥

  • keygen属性：challenge、keytype、autofoucus、name、disabled、form；

输入验证

确保有值

  • required，必须；

指定范围

  • min和max；

模式匹配

  • pattern，正则匹配；

检查 url 与 邮箱

  • 结合正则使用；

禁用输入验证

  • novalidate 和 formnovalidate；
<input type="submit" value="Save" foormnocalidate />

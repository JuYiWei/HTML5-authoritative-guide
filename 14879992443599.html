基本文本样式

对齐文本

  • text-align
属性 说明
start
end
left
right
center
justify
  • text-justify 添加空白方式；
值 说明
auto
none 禁用文本对齐
inter-word 适用英语
inter-ideograph 适用汉语日韩
inter-cluster 适用泰语
distribute ？
kashida 草体

处理空白

  • whitespace
值 说明
normal 空白压缩，文本换行
nowrap 空白压缩，文本不换行
pre 空白保留，文本遇到换行符时换行
pre-line 空白压缩，文本满行货换行符时换行
pre-wrap 空白保留，文本满行货换行符时换行

文本方向

  • direction
值 说明
ltr 左
rtl 右

文本间距

属性 说明
letter-spacing 字母间距
word-spacing 单词间距
line-height 行高

断词

  • word-wrap
值 说明
normal 单词不断
break-word 单词断开

首行缩进

  • text-indent

text-decoration 值 说明
none
underline
overline
line-through
blink
text-transform 值 说明
none
capitalize
uppercase
lowercase

文本阴影

text-shadow 值 说明
h-shadow
v-shadow
blur
color

选择字体

  • font-family 可指定多个字体，按优先级排序；

字体大小

  • font-size
值 说明
xx-small,x-small,small,medium,large,x-large,xx-large 指定字体
smaller,larger 相对父元素
指定数值
% 相对父元素的百分比

}

}

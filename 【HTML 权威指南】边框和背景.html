3.4 边框和背景

  • 边框样式
  • 元素背景
  • 盒子阴影
  • 应用轮廓

边框样式

边框属性 说明
border-width 边框宽度
border-style 边框样式
border-color 边框颜色 
p {
    border-width:5px;
    border-style:solid;
    border-color:red;
}
  • 宽度
值 说明
长度值
绘制区域的百分数
thin 细
medium 中等
thick 粗
  • 样式
值 说明
none 无
dashed 破折号
dotted 圆点
double 双线
groove 槽线
inset 内嵌效果
outset 突出效果
ridge 脊线
solid 实线
  • 单边样式
属性 说明
border-top-width 顶部，宽
border-top-style 顶部，样式
border-top-color 顶部，颜色
-bottom- 同上，底部
-left- 同上，左侧
-right- 同上，右侧

  • border 简写

  • border 宽 样式 颜色 空格分隔即可；

p {
    border:thin solid black;
}
  • 圆角
属性 说明
border-radius 四个圆角
border-top-left-radius 左上圆角，其他类似
  • 基本使用，注意使用 / 分隔；以实际需求再调整，可调节度高；
p {
    border: thin black solid;
    border-radius: 10px/20px;
}
  • 图像边框
属性 说明
border-image 简写用
border-image-source 图片来源
border-image-slice 切分图像的便宜
border-image-width 图像边框宽度
border-image-outset 向外拓展部分
border-image-repeat stretch、repeat、round 中的 1 个或 2 个
  • 切分图像
border-image: url("123.jpg") 10/50px;
  • 切分图控制方式
属性 说明
stretch 拉伸切分图填满
repeat 平铺
round 不截断，平铺
space 不截断，平铺，图片之间保留间距 
border-image: url("123.jpg") 10/50px round repeat;

元素背景

  • 背景颜色和图像
p {
    background-image: url("123.jpg");
    background-color: red;   
}
background-repeat 值 说明
repeat-x x 平铺，截断
repeat-y y 平铺，截断
repeat 平铺，截断
space 平铺，添加间隙，不截断
round 平铺，调整大小，不截断
no-repeat 不平铺
  • 背景填充方式
background-size 值 说明
contain 比例缩放，以长边为准
cover 比例缩放，以短边为准
auto 本身尺寸显示
  • 背景位置
background-position 值 说明
top 太明显了，略
left
right
bottom
center
  • 背景附着方式
background-attachment 值 说明
fixed 固定
local 跟随内容滚动
scroll 跟随元素滚动
  • 背景开始位置和裁剪方式
background-origin 和 background-clip 值 说明
border-box 边框盒子内绘制
padding-box 内边距盒子内绘制
content-box 内容盒子绘制
  • 简写方式
background:<color> <position> <size> <repeat> <origin> <clip> <attachment> <image>

盒子阴影

box-shadow 值 说明
hoffset 水平
voffset 竖直
blur 模糊
spread 延时
color 颜色
inset 内嵌阴影

应用轮廓

  • border 类似；
属性 说明
outline 简写
outline-color 颜色
outlin-offset 偏移
outline-style 样式
outlin-width 宽

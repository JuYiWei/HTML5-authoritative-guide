compatMode 读 文档兼容模式
cookie 读写 文档 cookie

location 读 文档 URL
readyState 读 文档状态
referrer 读 连接到文档的 URL
title 读写 文档标题

search 读写 URL 查询部分
hash 读写 URL 锚部分
assign 导航到指定的 URL
replace 清除当前文档并导航到 URL 所指的文档
reload 重载
resolveURL 将指定的相对 URL 解析成绝对 URL

属性 说明

body body 元素
Embeds plugins 所有 embed 元素

head head 元素
images 所有 image 元素
links 所有 link 元素
scripts 所有 script 元素

2. 数组标记获取已命名的元素

  • 略；

3. 搜索元素

属性 说明
getElementById 指定 id 元素，唯一
getElementByClassName 指定 class 元素集合
getElementByName 指定 name 元素集合
getElementByTagName 指定 类型 元素集合
querySelector 指定 CSS 选择器的第一个元素
querySelectorAll 指定 CSS 选择器的元素集合
  • getElemenetBy- 使用方便；
var elems = document.getElemenetByClassName("image");
// class="image"
  • querySelector- ：选择度更广更灵活；
var elems = document.querySelectorAll("p.image")
// p 元素且 class="image"

  • 合并进行链式搜索

  • 对于上述搜索方法，可以进行二次过滤；除 getElemenetById 只能由 document 对象开始；

var elems = document.getElemenetByClassName("image").getElementByName("appleIcon");
// class="image" 且 name="appleIcon"

属性 说明
parentNode 父元素
hasChildNodes() 当前元素有子元素
childNodes 子元素集合
firstChild 第一个子元素
lastChild 最后一个子元素
previousSibling 上一个兄弟元素
nextSibling 下一个兄弟元素

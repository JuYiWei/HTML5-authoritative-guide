Doucument 元数据

属性 说明
characterSet 读 文档的字符集编码
charset 读写 文档的字符集编码
compatMode 读 文档兼容模式
cookie 读写 文档 cookie
defaultCharset 读 浏览器默认字符集编码
defaultView 返回 Window 对象
dir 读写 文档文本方向
domain 读写 文档域名
implementation 读 可用 DOM 功能信息
lastModified 读 文档修改最后时间
location 读 文档 URL
readyState 读 文档状态
referrer 读 连接到文档的 URL
title 读写 文档标题

location 对象

属性 说明
protocol 督学 URL 协议
host 读写 URL 的主机和端口
href 读写 文档地址
hostname 读写 URL 主机名称
port 读写 URL 端口
pathname 读写 URL 路径
search 读写 URL 查询部分
hash 读写 URL 锚部分
assign 导航到指定的 URL
replace 清除当前文档并导航到 URL 所指的文档
reload 重载
resolveURL 将指定的相对 URL 解析成绝对 URL

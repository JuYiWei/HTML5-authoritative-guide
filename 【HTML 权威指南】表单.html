一、基本表单

<form method="post" action="http://xxxx:8080/login">
    <label>用户名：<input name="name"/></label>
    <label>密码：<input name="password"/></label>
    <button>login</button>
</form>

二、配置表单

1. action

  • action：提交表单服务器 url；
<form action="http://xxxxx:8080/form>

2. HTTP 方法

  • method：GET 和 POST；
<form method="post">

3. 配置数据编码

  • enctype

|值|说明|
|application/x-www-form-urlencode|默认编码方式，不能上传文件|
|multipart/form-data|上传文件|
|text/plain|与浏览器有关，建议不要使用|

4. 表单自动完成

  • autocomplete：对form和input都可以设置；

5. 提交表单之后的打开方式

  • target：与a的target一样；
值 说明
_blank 新窗口
_parent 父窗口
_self 当前
_top 顶层
指定

6. 设置表单名称

  • name属性，与id不同，方便使用 DOM；form的name属性不会上传，input的name属性，会当成 key，用户内容为 value 上传；

三、说明标签

  • label
<label>密码：<input name="password"/></label>

四、自动聚焦 input

  • autofocus；
<label>用户名：<input autofocus name="name"/></label>

五、禁用 input

  • disabled；
<label>验证码<input name="code" disabled></label>

六、表单元素编组

1. fieldset;

<form method="post" action="http://xxxx:8080/register" autocomplete="off">

    <fieldset>
        <label>用户名：<input autofocus name="name"/></label>
        <label>密码：<input name="password"/></label>
        <label>验证码：<input name="code" disabled></label>
    </fieldset>

    <fieldset>
        <label>地址：<input name="address"></label>
    </fieldset>

    <button>register</button>
</form>

2. 说明标签

  • legend；
<form method="post" action="http://xxxx:8080/register" autocomplete="off">

    <fieldset>
        <legend>用户信息</legend>
        <label>用户名：<input autofocus name="name"/></label>
        <label>密码：<input name="password"/></label>
        <label>验证码：<input name="code" disabled></label>
    </fieldset>

    <fieldset>
        <legend>基本信息</legend>
        <label>地址：<input name="address"></label>
    </fieldset>

    <button>register</button>
</form>

3. fieldest 禁用 input 元素

<form method="post" action="http://xxxx:8080/register" autocomplete="off">

    <fieldset>
        <legend>用户信息</legend>
        <label>用户名：<input autofocus name="name"/></label>
        <label>密码：<input name="password"/></label>
        <label>验证码：<input name="code" disabled></label>
    </fieldset>

    <fieldset disabled>
        <legend>基本信息</legend>
        <label>地址：<input name="address"></label>
    </fieldset>

    <button>register</button>
</form>

七、button 元素

值 说明
submit 提交表单
reset 重置
buton 无

1. 提交表单

  • submit类型时，额外属性；
属性 说明
form 关联表单
formaction 重新关联 action
formenctype 重新关联 enctype
formmethod 重新关联 method
formtarget 重新关联 target
formnovalidate 重新关联，与输入数据检查有关 
<button type="submit" formaction="http://xxxxx:8443/regisert">register</button>

2. 重置表单

  • reset，重置表单输入内容；
<button type="reset">Reset</button>

3. 一般按钮使用

  • button，可用于触发 JavaScript；
<button type="button">Normal Button</button>

八、表单外使用表单元素

  • input和button可以解除必须在form元素内部的限制，只有使用form属性关联即可；
<form method="post" action="http://xxxx:8080/register" id="registerForm">
    <fieldset>
        <legend>用户信息</legend>
        <label>用户名：<input autofocus name="name"/></label>
        <label>密码：<input name="password"/></label>
    </fieldset>
</form>

<button type="reset" form="registerForm">Reset</button>

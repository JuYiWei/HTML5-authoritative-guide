一、输入文字

  • type 类型为 text，拥有额外属性；
text 额外属性 说明
dirname 方向
list 建议输入值
maxlength 字符最大数量
pattern 正则表达式
placeholder 提示
readonly 只读
required 必须输入
size 宽度
value 初始值

1. 大小

  • maxlength：输入字符个数限制；
  • size：可显示字符数限制，宽度限制；
<label>用户名：<input name="name" maxlength="8"/></label>
<label>密码：<input name="password" size="10"/></label>

2. 初始值与占位提示

  • value：初始值；
  • placeholder：占位提示；
<label>用户名：<input name="name" value="Li Lei"/></label>
<label>密码：<input name="password" placeholder="123456"/></label>

3. 数据列表

<label>用户名：<input name="name" list="userNames"/></label>
    
<datalist id="userNames">
    <option value="Li Lei"></option>
    <option value="Han Meimei"></option>
    <option value="Xiao Ming"></option>
</datalist>

4. 只读或禁用

  • readonly：只读；
  • disable：禁用；
<label>用户名：<input name="name" readonly/></label>
<label>密码：<input name="password" disabled/></label>

5. 方向

  • 略；

二、输入密码

  • type 类型为 password，额外属性与text用法一样；
text 额外属性 说明
maxlength 字符最大数量
pattern 正则表达式
placeholder 提示
readonly 只读
required 必须输入
size 宽度
value 初始值

三、生成按钮

  • submitresetbutton，会对应变成button的三种类型，使用方式一样；

四、数据检查

|属性|说明|
|checkbox|是否选择|
|color|颜色|
|date|日期|
|datetime|带时区的世界时|
|datetime-local|不带时区的世界时|
|email|油箱|
|month|年月|
|number|数字|
|radiobutton|一组固定选项中选择|
|range|指定范围|
|tel|电话|
|time|时间|
|week|年与星期|
|url|URL|

1. 数值

  • number
属性 说明
list 建议值
min 最小值
max 最大值
readonly 只读
required 必须
step 数值调节步长
value 初始值 
<label>购买数量：<input type="number" min="0" max="10" step="2"></label>

2. 指定范围

  • range，与number功能相同，显示不同；
<label>购买数量：<input type="range" min="0" max="10" step="2"></label>

3. 布尔、多选

  • checkbox
属性 说明
checked 选中
min 必须
max 勾选状态时提交的数据值，默认 on 
<label>apple：<input type="checkbox" name="fav" checked></label>

4. 单选

  • radio，属性与checkbox一致；
<label>性别：
    <input type="radio" name="sex" value="man">男
    <input type="radio" name="sex" value="woman">女
</label>

5. 规定格式的字符串

  • email、tel、url；
属性 说明
list 选择列表
maxlength 字符限制
pattern 正则
placeholder 占位提示
readonly 只读
required 必须
size 宽度
value 初始值

email 还有额外的multipe属性，接受多个邮箱地址； 

<label>邮箱：<input type="email"></label>

6. 时间日期

  • 直接使用 jQuery 之类的 JS 日历选择工具好了；

7. 颜色值

  • color，
<label>背景颜色：<input type="color" name="backgroundColor"></label>

五、获取搜索用词

  • search，一般浏览器都直接以text类型处理；但是可以根据不同类型有不同的样式；

六、生成隐藏的数据项

  • hidden，用户不需要输入，也不给用户看到，但是默认上传给服务器；
<input type="hidden" name="token" value="qs1234">

七、生成图像按钮和分区响应图

  • image，作用同button类型，属性稍多几个；
<input type="image" src="123.jpg" name="submit">
属性 说明
alt 说明文字
formaction 重新关联 action
formenctype 重新关联 enctype
formmethod 重新关联 method
formtarget 重新关联 target
formnovalidate 重新关联，与输入数据检查有关
height 高
src 图片资源
width 宽

八、上传文件

  • file；
<input type="file" accept="application/pdf" required>
属性 说明
accept MIME 类型
multiple 文件多选，浏览器支持不多？
required 必须

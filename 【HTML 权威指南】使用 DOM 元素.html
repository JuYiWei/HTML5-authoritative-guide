属性 说明
add 添加类

toggle 不存在就添加，存在就移除

2. 使用元素属性

属性 说明
attributes 元素属性集合
dataset data- 开头的属性

hasAttribute() 判断指定属性
removeAttribute() 移除指定属性
setAttribute(,<value) 设置指定属性

  • 文本对象，本身没有标签，难以定位，只能通过父元素向下获取；

splitText 切分字符串
substringData, 子字符串
wholeText 获取文本

三、修改模型

1. 创建和删除元素

属性 说明
createElement() 创建元素
createTextNode() 创建文本
removeChild() 移除元素 
<ol id="myol">
    <li id="li1">1111</li>
    <li>2222</li>
</ol>


<script>
    var olElement = document.getElementById("myol");

    var newli = document.createElement("li");
    olElement.appendChild(newli);

    var newlitext = document.createTextNode("3333");
    newli.appendChild(newlitext);

    var li1 = document.getElementById("li1");
    olElement.removeChild(li1);
</script>

2. 复制元素

属性 说明
cloneNode(boolean) 复制元素，是否复制其子元素

3. 移动元素

属性 说明
appendChild() 拼接元素 
<ol id="myol1">
    <li>1111</li>
    <li>2222</li>
</ol>

<ol id="myol2">
    <li>aaa</li>
    <li id="flag">333</li>
</ol>


<script>
    var olElem= document.getElementById("myol1");
    var liElem = document.getElementById("flag");

    olElem.appendChild(liElem);
</script>

4. 比较元素对象

属性 说明
isEqualNode() 是否元素内容一样
isSameNode() 是否同一个元素

5. 使用 HTML 片段

属性 说明
innerHTML 读写某元素的内容
outerTHML 读写某元素的 HTML 和内容
insertAdjacetHTML(,) 相对于元素 pos 插入 HTML
insertBefore(,) 在 childlem 元素前插入 newElem
pos 值 说明
afterbenin 作为当前元素的第一个子元素插入
beforeend 作为当前元素的最后一个子元素插入
afterend 插入当前元素之后
beforebegin 当前元素之前

6. 向文本块插入元素

<p id="p1">There are lots of different kinds of fruit</p>

<script>
    var pElem= document.getElementById("p1");
    pElem.firstChild.splitText(10);
    var newText = pElem.childNodes[1].splitText(4).previousSibling;    
    pElem.insertBefore(document.createElement("i"),newText).appendChild(newText);
</script>

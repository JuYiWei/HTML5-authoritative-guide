嵌入图像

<img src="123.jpg" alt="image" width="300px" height="300px">

超链接图片

<a href="Home.html"><img src="123.jpg"></a>

分区响应图

  • map
<img src="123.jpg" width="400px" height="400px" usemap="#mymap">

<map name="mymap">
    <area href="Home.html" shape="rect" coords="0,0,100,100">
    <area href="Record.html" shape="rect" coords="300,300,100,100">
</map>

嵌入另一个 HTML 文档

  • iframe，安全限制，需要手动开启内部的表单、脚本等；
<a href="Record.html" target="myframe">asd</a>

<iframe name="myframe" src="Home.html" width="300" height="300" sandbox="allow-script">

插件嵌入内容

  • 比较灵活，简单介绍，没有深入研究；

embed 插件元素

<embed src="Record.html" type=""></embed>

object 插件元素

<object data="Record.html"><param name="" value=""></object>

object元素的其他用途

  • 感觉并没什么软用，这些其他用途；

嵌入图像

<object data="123.jpg"></object>

图片分区控制

<object data="123.jpg" type="image/png" usemap="#mymap"></object>

作为上下文

<object type="text/html" name="frame"></object>

嵌入数字表达式

显示进度

<progress id="myprogress" value="10" max="100"></progress>
<button onclick="complete()">click</button>

<script>
    function complete() {
        var press = document.getElementById("myprogress");
        press.value = press.value+10;
    }
</script>

显示范围的值

  • 跟进度条类似，但是不同进度不同颜色；
<meter id="mymeter" value="10" min="0" max="100" low="60" high="90"></meter>
<button onclick="complete()">click</button>

<script>
    function complete() {
        var mymeter = document.getElementById("mymeter");
        mymeter.value = mymeter.value+10;
    }
</script>

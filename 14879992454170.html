结构性伪类

根元素

  • :root，总是返回html元素，实际使用可能比较少；
:root {
    border: thin black solid;
    background: grey;
}

子元素

选择器 说明
:first-child 第一个子元素
:last-child 最后一个子元素
:only-child 唯一子元素
:only-of-type 唯一指定类型的子元素
  • first-child 和 last-child是一样的；
// 每个元素的第一个子元素；
:first-child {
    border: thin black solid;
}

// 第一个子元素，且为 p 元素
p:first-child {
    color: yellow;
}

// body 中的第一个子元素，且为 p 元素
body>p:first-child {
    color: red;
}

// 以上三种情况，用下面的看比较直观
<body>
<p>body 下第 1 个 p</p>
<p>body 下第 2 个 p</p>
<div>
    <p>div1 下第 1 个 p</p>
    <p>div1 下第 2 个 p</p>
</div>
<div>
    <h1>div2 下第 1 个 h1</h1>
    <p>div2 下第 1 个 p</p>
    <p>div2 下第 2 个 p</p>
</div>
</body>
  • only-child 唯一子元素；
// 唯一子元素
:only-child {
    color: red;
}

// 唯一子元素，且为 p 元素
p:only-child {
    color: red;
}

// div 中唯一子元素，且为 p 元素
div>p:only-child {
    color: red;
}

// 以上三种情况，用下面的看比较直观
<body>
<p>body 下第 1 个 p</p>
<div>
    <p>div1 下唯一 p</p>
</div>
<div>
    <p>div2 下第 1 个 p</p>
    <p>div2 下第 2 个 p</p>
</div>
<span>
    <p>span 下唯一 p</p>
</span>
</body>
  • only-of-type 唯一指定类型子元素；
// 唯一类型子元素
:only-of-type {
    border: thin black solid;
}

// 唯一类型子元素，且为 p 元素
p:only-of-type {
    border: thin black solid;
}

// div 元素下的唯一类型子元素，且为 p 元素
div>p:only-of-type {
    border: thin black solid;
}

// 以上三种情况，用下面的看比较直观
<body>
<p>body 下唯一 p</p>
<div>
    <h1>div1 下唯一 h1</h1>
    <p>div1 下唯一 p</p>
</div>
<div>
    <p>div2 下第 1 个 p</p>
    <p>div2 下第 2 个 p</p>
</div>
<span>
    <p>span 下唯一 p</p>
</span>
</body>

nth-child

选择器 说明
:nth-child(n) 第n个子元素
:nth-last-child(n) 倒数第n个子元素
:nth-of-type(n) 指定类型的第n个子元素
:nth-last-of-type(n) 指定类型的倒数第n个子元素

与上述类似，略了，要举一反三；

UI 伪类

启用/禁用

  • :enabled 和 :disabled
:enabled {
    color: red;
}

:disabled {
    color: green;
}


<button>可用</button>
<button disabled>不可用</button>

已勾选

  • :checked

默认

  • :default

有效/无效的 input

  • :valid 和 :invalid

限定范围的 input

  • :in-range 和 out-of-range

必须/可选的 input

  • :required 和 :optional

动态伪类

link 超链接

visited已访问的超链接

hover 鼠标悬停

active 元素激活时

focus 焦点元素

其他伪类

not 否定选择器

empty 没有子元素的元素

lang 基于 lang 全局属性的元素

target 超链接 target 属性相关

:target {
    border: thin black solid;
    color: red;
}


<a href="#p1">p1</a>
<a href="#p2">p2</a>

<p id="p1">content 1</p>
<p id="p2">content 2</p>

基本选择器

所有元素

* {
    border: thin black solid;
}

元素类型

p{
    color:blue;
}

类

// 所有
.mygreen {
    color: green;
}

// 指定 p 类型
p.myred {
    color:red;
}

<p class="myred">CSS 选择器</p>

ID

#myblue {
    color: blue;
}

<p id="myblue">CSS 选择器</p>

属性

[href] {
    color: orange;
}

<a href="Record.html">记录查询</a>
条件 说明
[attr] attr 属性，忽略其值
[attr=val] attr 属性，且值为 val
[attr=val] attr 属性，且以字符串 val 开头
[attr$=val] attr 属性，且以字符串 val 结尾
[attr*=val] attr 属性，且包含字符串 val
[attr~=val] attr 属性，且有多个值，其中一个是 val
[attr =val]

复合选择器

并集

  • 逗号分割即可；
a, [lang|="en"] {
    border: thin black solid;
}

后代

// 选择 id 为 mytable 的td 元素
#mytable td {
    border: thin black solid;
    color: red;
}


<table id="mytable">
    <tr>
        <th>name</th>
        <th>age</th>
    </tr>
    <tr>
        <td>Li Lei</td>
        <td>18</td>
    </tr>
</table>

子元素

  • 直接后代；
div > span {
    color: red;
}


<div>
    <span>111</span>
    <p>
        <span>222</span>
    </p>
</div>

兄弟元素

  • 相邻的兄弟元素；
// 注意，这里匹配的是后者 a，不是 2 个都是；
p + a {
    color: red;
}


<p class="mygreen">今天天气不错</p>

p ~ a {

伪类选择器

::first-line

p::first-line {
    color: red;
}


<p class="mygreen">今天天气不错<br>??? </p>

::first-letter

p::first-letter {
    color: red;
}


<p>今天天气不错<br>??? </p>

:before 和 :after

  • 使用 content 值，分别在元素前后插入内容；
a:before {
    content: "Click here to";
}

a:after {
    content: "!";
}

CSS 计数器

body {
    counter-reset: myacount;
}

p:before {
    content: counter(myacount)".";
    counter-increment: myacount;
}

p:after {
    content: "；";
}

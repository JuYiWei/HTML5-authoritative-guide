一、使用简单事件处理器

1. 实现简单的内联事件处理器

  • 实际使用，并不方便；
    <style>
        p {
            color: red;
            background-color: green;
        }
    </style>
</head>

<p onmousemove="this.style.backgroundColor='red';this.style.color='green'" id="p1">There are lots of different kinds of fruit</p>
  • 实现一个简单的事件处理函数
<style>
    p {
        color: black;
        background-color: white;
    }
</style>


<script>
    function pMouseOver(elem) {
        elem.style.backgroundColor = 'black';
        elem.style.color = 'white';
    }

    function pMouseOut(elem) {
        elem.style.backgroundColor = 'white';
        elem.style.color = 'black';
    }
</script>

<p onmousemove="pMouseOver(this)" onmouseout="pMouseOut(this)" id="p1">There are lots of different kinds of fruit</p>

二、使用 DOM 和事件对象

1. 使用默认的 on 开头的系统方式；

<p id="p1">There are lots of different kinds of fruit</p>

<script>
    var pElem = document.getElementById('p1');
    pElem.onmouseover = pMouseOver;
    pElem.onmouseout = pMouseOut;
    
    function pMouseOver(elem) {
        elem.target.style.backgroundColor = 'black';
        elem.target.style.color = 'white';
    }

    function pMouseOut(elem) {
        elem.target.style.backgroundColor = 'white';
        elem.target.style.color = 'black';
    }
</script>

注意不是直接使用函数，而是注册函数监听；并且脚本必须在 DOM 之后执行；

  • addEventListener 和 removeEventListener；
<script>

    var pElem = document.getElementById('p1');
    pElem.addEventListener("mouseover", pMouseOver);
    pElem.addEventListener("mouseout", pMouseOut); // 原来是 onmouseout 此处不同；

    function pMouseOver(elem) {
        elem.target.style.backgroundColor = 'black';
        elem.target.style.color = 'white';
    }

    function pMouseOut(elem) {
        elem.target.style.backgroundColor = 'white';
        elem.target.style.color = 'black';
    }

</script>
Event 属性 说明
type 事件名称
target 事件指向的元素
currentTarget 当前被触发事件监听器的元素
eventPhase 事件生命周期的阶段
bubbles 事件是否会在文档里冒泡
cancelable 事件是否有默认的撤销
timeStamp 时间戳
stopPropagation 当前元素的事件监听器触发后终止事件在元素树中流动
stopImmediatePropagation 立即终止事件在元素树中流动
preventDefault 防止浏览器执行与事件关联的默认操作
defaultPrevented 如果调用过 preventDefault()则返回 true

2. 按类型区分事件

<script>
    var pElem = document.getElementById('p1');
    pElem.onmouseover = handleMouseEvent;
    pElem.onmouseout = handleMouseEvent;

    function handleMouseEvent(e) {
        if (e.type == "mouseover") {
            e.target.style.backgroundColor = 'black';
            e.target.style.color = 'white';
        } else if (e.type == "mouseout") {
            e.target.style.backgroundColor = 'white';
            e.target.style.color = 'black';
        }
    }
</script>

2. 理解事件流

属性 说明
CAPTURING_PHASE 捕捉
AT_TARGET 目标
BUBBLING_PHASE 冒泡

  • 捕捉：找出事件涉及元素，检查目标元素与 body 元素之间的所以元素，优先处理他们的事件处理器，然后是目标的处理器；

  • 目标：执行目标元素的添加的事件；

  • 冒泡：从目标元素，沿父元素向 body 元素检查，检查存在该事件类型但未启用捕捉监听器；可以用 bubbles 属性检查事件是否支持冒泡；

3. 使用可撤销事件

  • preventDefault 不会阻止事件流的捕捉、目标、冒泡，但是不会执行默认行为；
<a href="https://www.baidu.com">baidu</a>

<script>
    function handleAClick(e) {
        if (!confirm("ok?")) {
            e.preventDefault();
        }
    }
    
    var es = document.querySelectorAll("a");
    for (var i=0;i<es.length;i++) {
        es[i].addEventListener("click", handleAClick, false);
    }
</script>

三、使用 HTML 事件

1. 文档和窗口事件

  • Document 对象的事件
属性 说明
readystatechange 在 readyState 属性变化时触发
  • Window 对象的事件
属性 说明
onabort 文档或资源加载过程被终止时触发
onafterprint 已调用 Window.print() 方法，但尚未给用户提供打印选项时触发
onbeforeprint 用户完成打印后触发
onerror 文档或资源的加载发生错误是触发
onhashchange 锚部分发生变化时触发
onload 文档或资源加载完成时触发
onpopstate 触发后提供一个关联浏览器历史的状态对象
onresize 窗口缩放时触发
onunload 文档从窗口关闭时触发

2. 使用鼠标事件

属性 说明
click 点击释放时
dblclick 双击释放时
mousedown 点击时
mouseenter 鼠标移入某元素区域时
mouseleave 鼠标移除某元素区域时
mousemove 鼠标在某元素上移动时
mouseout 与 mouseleave 类似
mouseover 与 mouseenter 类似
mouseup 鼠标按键释放时
MouseEvent 属性 说明
button 表示点击的是哪个键 0-主键，1-中键，2-次键
altKey 事件触发时按下了 alt/option 键
clientX 触发事件时鼠标相对元素的 X
clientY 触发事件时鼠标相对元素的 Y
screenX 触发事件时鼠标相对屏幕的 X
screenY 触发事件时鼠标相对屏幕的 Y
shiftKey 事件触发时按下了 Shift
ctrlKey 事件触发时按下了 Ctrl

focusout 即将丧失焦点时
FocusEvent 对象 说明
relatedTarget 即将获得或者失去焦点的元素

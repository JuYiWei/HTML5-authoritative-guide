超链接

属性 说明
href 资源 URL
hreflang 资源语言
media 与第七章 link 中相同
rel 与第七章 link 中相同
target 打开方式
type MEMI 类型

外部超链接

<a href="https://www.baidu.com">baidu</a>

相对 URL

  • 根据当前页面的 URL 确定目标资源；
<a href="Setting.html">baidu</a>

内部超链接

// 元素设置 id 
<p id="page2">第二页</p>

// 链接到指定 id 位置
<a href="#page2">baidu</a>

指定浏览器环境

target 属性 说明
_blank 新建窗口
_parent 父窗框组
_self 当前，默认
_top 顶层窗口
指定窗框

基本文字元素标记内容

关键词

<b>bbbbbbbbbbbbbbbb</b>

强调

<em>emememememememem</em>

与周围不同

<i>iiiiiiiiiiiiiii</i>

不准确或校正

<s>sssssssssssssss</s>

重要

<strong>strongstrong</strong>

下划线

<u>uuuuuuuuuuuuuuuuu</u>

小号字体

<small>smallsmall</small>

上标下标

<sub>sub</sub>
<sup>sup</sup>

换行

强制换行 br

  • 直接换行；

建议换行 wbr

  • 浏览器会判断，是否需要换行；

输入输出

略

<code>code</code>
<var>var</var>
<samp>samp</samp>
<kbd>kbd</kbd>

标题引用、引文、定义、缩写

缩写

<abbr title="Photoshop">PS</abbr>

定义

<abbr title="爬山">PS</abbr>

引文

<q>你这样还活的下去，租房加吃饭都嫌勉强了。所以我不在广州租房，这几年长期外派出差，公司管住另外每天给一百多点的出差补贴，要不然呢。</q>

标题引用

<cite>你这样还活的下去，租房加吃饭都嫌勉强了。所以我不在广州租房，这几年长期外派出差，公司管住另外每天给一百多点的出差补贴，要不然呢</cite>

语言元素

ruby、rt、rp

  • 汉字加拼音（支持该属性的rp标签不显示，反之显示，布局也不同）；
<ruby>魑<rp>(</rp><rt>chi</rt><rp>)</rp></ruby>

bdo

  • 内容文字方向
<bdo dir="ltr">left</bdo>
<bdo dir="rtl">right</bdo>

bdi

  • 包含一些奇葩的文字，例如阿拉伯文字对排版引起的问题，可以用bdi处理；

其他文本元素

一般内容

<span>span</span>

突出显示文本

<mark>mark</mark>

表示添加和删除的内容

<ins>ins</ins>
<del>del</del>

表示时间和日期

<time datetime="2000-11-11">December 5th</time>
